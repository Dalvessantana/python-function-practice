## 📖 Sobre o Projeto

Este repositório reúne exercícios práticos desenvolvidos para estudar e consolidar os fundamentos de **funções em Python**.

Durante os exemplos são abordados conceitos essenciais da linguagem, como criação de funções, passagem de parâmetros, retorno de valores, reutilização de código e interação com o usuário.

O objetivo principal é compreender como utilizar funções para tornar programas mais organizados, legíveis e reutilizáveis.

## 📚 Conteúdo

Os exercícios abordam:

- Criação de funções
- Parâmetros e argumentos
- Retorno de valores
- Reutilização de código
- Entrada de dados com `input()`
- Manipulação de strings
- Conversão de temperaturas
